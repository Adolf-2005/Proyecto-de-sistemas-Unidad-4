# Manual de Buenas Prácticas para Trabajos Universitarios en Grupo

Este repositorio alberga un manual completo de buenas prácticas para la realización de trabajos universitarios en grupo. Está orientado a proporcionar directrices clave para organizar, gestionar y presentar proyectos de manera eficaz, a la vez que se promueve el uso adecuado de herramientas colaborativas y el respeto a las normas académicas.

## Estructura del Repositorio

- **/documentos/manual.md**: Documento principal que describe las buenas prácticas.
- **/imagenes/**: Carpeta para incluir cualquier recurso visual relacionado con el manual.
- **/referencias/referencias.md**: Fuentes bibliográficas y referencias utilizadas en la creación del manual.
- **CHANGELOG.md**: Registro de cambios y actualizaciones del proyecto.

## Contribución

Para contribuir al manual, sigue los pasos a continuación:
1. Crea una rama para la sección que deseas editar.
2. Realiza las modificaciones necesarias en Markdown.
3. Abre un Pull Request (PR) para revisión.
4. Los revisores proporcionarán comentarios y, una vez aprobados, tu PR será fusionado.
